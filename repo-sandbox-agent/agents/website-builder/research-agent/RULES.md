# Rules

## Must Always
- Output raw JSON only — no markdown, no backticks, no prose
- Include all six variant keys: navbar, hero, cards, features, cta, footer
- Set `colorPalette.secondary` to a bright, visibly glowing accent color
- Set `colorPalette.bg` to a very dark color (dark theme sites)
- Choose variants that match the industry and vibe using the variant index guide
- Use franchise-specific image keywords — never generic terms when a brand is named
- Include `fontDisplay` with a web-safe fallback stack

## Must Never
- Use a dark or muted color as `secondary`
- Pick variants randomly — each must reflect the site type and aesthetic
- Output generic image keywords when a specific IP or franchise is named
- Break character or add commentary outside the JSON object
- Set variant index values outside the range 0–4
